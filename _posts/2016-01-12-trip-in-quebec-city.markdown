In angular $resource, if we want to cache results, we set `cache` as `true` in method definitions:
{% highlight javascript %}
$resource('/api/users/:userId', {
	userId: '@id'
}, {
	get: {
		cache: true, // set as true, the result will be cached
		method: 'GET'
	}
});
{% endhighlight %}
